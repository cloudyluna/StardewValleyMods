	\subprojecttitle{\subprojectname}{
		Allow crab pots to collect aquatic jellies.
		
		This mod uses the same trash spawn chance value to collect jellies randomly. The spawn type and in which area are as follow:
		
			\begin{itemize}
				\item Sea Jelly $\longrightarrow$ Ocean tiles i.e by Willy's shop.
				\item River Jelly $\longrightarrow$ Rivers, lakes, sewers, etc.
				\item Cave Jelly $\longrightarrow$ The same as river jelly, but a bit rarer.
			\end{itemize}
	}
	
	\subprojectfeatures{
		\item Allow crab pots to collect aquatic jellies.
		\item An option to replace all trash that will be collected by crab pots with jellies. Configurable with Generic Mod Config Menu.
	}
